배틀 모드 핵심 구현 원리
옵션 선택 시스템: 게임 대기 화면에서 **'인원수(2~4인)'**와 **'출제 방식(A/B)'**을 먼저 선택하고 시작하도록 설계했습니다.

독립된 플레이어 두뇌(State) 분리: 기존에는 게임 전체를 관리하는 두뇌가 1개였지만, 이제는 1p~4p가 각각 독립적인 점수, 타이머, 오답 체크 두뇌를 가지도록 코드를 완전히 분리했습니다. 아이들이 동시에 화면을 마구 터치해도 절대 오류가 나지 않습니다.

출제 방식(A/B) 알고리즘:

A방식 (개별 랜덤): 플레이어마다 낱말이 무작위로 나옵니다. 서로 커닝이 불가능하여 오직 자신의 문제에만 집중해야 합니다.

B방식 (동일 출제): 모든 플레이어가 똑같은 1번 문제, 똑같은 2번 문제를 풉니다. (단, 보기 위치는 섞임) 먼저 맞히고 다음 문제로 넘어가는 **'진정한 스피드 레이싱'**의 쾌감을 느낄 수 있습니다.

전자칠판 꽉 찬 분할 화면(Flexbox): 선택한 인원수에 맞춰 화면 세로 기둥이 자동으로 2등분~4등분 되며, 각각 고유의 동물 아바타(🐶, 🐱, 🐰, 🐹)와 테마 색상을 부여해 자기 자리를 헷갈리지 않게 했습니다.
